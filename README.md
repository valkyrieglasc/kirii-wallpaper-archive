# Wallpaper Archive

A retro-styled wallpaper gallery built with plain HTML, CSS, and JavaScript.

</div>

---

## Features

- Browse wallpapers in a responsive grid
- Search the archive by name or link
- Open a larger preview by clicking an image
- Download wallpapers through the action button
- Read an embedded note panel with draggable behavior

---

## Structure

- index.html: main page layout
- styles.css: visual styling and theme
- script.js: gallery rendering and interactions
- note-content.html: note content shown in the panel

---

## Run locally

1. Clone the repository
2. Open index.html in your browser, or serve the folder with:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000/

---

## Notes

This project uses a remote data source for the wallpaper list and is designed with a dark, terminal-inspired aesthetic.
